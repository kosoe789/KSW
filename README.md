ငါ့မှာဖော်ပြပါ website တခုကို github မှာ တင်ထားပါတယ် ။ အခု ငါက ထပ်တိုးပြင်ခြြင်တယ်။ လိုချင်တာက လက်ရှိရှိနေတဲ့ ပုံစံအတိုင်းပဲ နောက်ထပ် နှစ်အလိုက် အလားတူ csv file တွေ ထပ်တင်ပြီး အဲဒီနှစ်အလိုက် dropdown list ကို ရွေးချယ်ပြီး ကြည့်ချင်တယ်။ အခုပါတာက တနှစ်စာအတွက်ပဲဖြစ်နေတယ်။ အခု ငါ ဥပမာအနေနဲ့ ထပ်ထည့်ချင်တာက ၂၀၂၃ နဲ့ ၂၀၂၆ ခုနှစ်အတွက် ထည့်ချင်တယ်။ အဲလိုရ‌အောင် ကုတ်ကို ပြင်ပေးပါ။

ဟုတ်ကဲ့ပါ၊ ၂၀၂၃၊ ၂၀၂၅ (လက်ရှိ) နဲ့ ၂၀၂၆ ခုနှစ်အလိုက် CSV ဖိုင်တွေကို Dropdown list ကနေ ရွေးချယ်ကြည့်ရှုနိုင်ဖို့အတွက် HTML ကုတ်ကို ပြင်ဆင်ပေးထားပါတယ်။

ဒီလိုအလုပ်လုပ်နိုင်ဖို့အတွက် GitHub Repository ထဲမှာ Folder Structure ကို အောက်ပါအတိုင်း ခွဲပြီး ဖိုင်တွေ ဆောက်ပေးဖို့ လိုအပ်ပါမယ် (ဒါမှမဟုတ် ကုတ်ထဲက လမ်းကြောင်းအတိုင်း ပြောင်းပေးရပါမယ်)။

📂 GitHub Folder ဖွဲ့စည်းပုံ အကြံပြုချက်
လက်ရှိ ဖိုင်တွေကို 2023, 2025, 2026 ဆိုပြီး Folder အလိုက် ခွဲထည့်ပေးလိုက်တာက အသန့်ရှင်းဆုံးနဲ့ စီမံရအလွယ်ဆုံး ဖြစ်သွားပါလိမ့်မယ်။

Plaintext

    GitHub မှာ Folder ဖွဲ့နည်းက သီးသန့် "Create Folder" ဆိုတဲ့ Button မရှိလို့ စတင်သုံးစွဲသူတွေအတွက် အနည်းငယ် ရှုပ်ထွေးနိုင်ပါတယ်။ GitHub က ဖိုင်တစ်ခုခု ဆောက်မှသာ Folder ကို အလိုအလျောက် တည်ဆောက်ပေးတဲ့စနစ် ဖြစ်ပါတယ်။

အလွယ်ကူဆုံး နည်းလမ်း (၂) ခုကို အောက်ပါအတိုင်း ရှင်းပြပေးထားပါတယ်ခင်ဗျာ။

နည်းလမ်း (၁) - GitHub Website ပေါ်မှ တိုက်ရိုက် Folder ဆောက်နည်း
သင်ပြောင်းလဲချင်တဲ့ GitHub Repository (KSW) ထဲကို အရင်ဝင်ပါ။

ညာဘက်အပေါ်နားက "Add file" ကိုနှိပ်ပြီး "Create new file" ကို ရွေးပါ။

"Name your file..." ဆိုတဲ့ ကွက်လပ်မှာ မိမိပေးချင်တဲ့ Folder နာမည်ကို ရိုက်ထည့်ပါ (ဥပမာ - 2023)။

ပြီးရင် ကီးဘုတ်က Slash / လက္ခဏာကို ရိုက်နှိပ်လိုက်ပါ။ ထိုအခါ သင်ရိုက်လိုက်တဲ့ နာမည်က Folder ပုံစံအဖြစ် အလိုအလျောက် ပြောင်းလဲသွားပါလိမ့်မယ်။

GitHub စနစ်အရ Folder ထဲမှာ အနည်းဆုံး ဖိုင်တစ်ခုရှိမှ သိမ်းဆည်းလို့ရမှာဖြစ်လို့ ဖိုင်နာမည်နေရာမှာ Level1A.csv သို့မဟုတ် ယာယီဖိုင်အဖြစ် readme.md (သို့) .keep လို့ ရိုက်ထည့်ပါ။

အောက်ဆုံးသို့ ဆင်းပြီး "Commit changes..." (စိမ်းပြာရောင်ခလုတ်) ကို နှိပ်ပြီး အတည်ပြုလိုက်ပါ။

💡 မှတ်ချက်။ GitHub ပေါ်မှာ အထဲမှာ ဘာဖိုင်မှမရှိတဲ့ Folder အလွတ် (Empty Folder) သီးသန့် ဆောက်လို့မရပါဘူး။ အနည်းဆုံး ဖိုင်တစ်ခုခု ရှိနေရပါမယ်။


🛠 GitHub Website ပေါ်တွင် ဖိုင်ရွှေ့နည်းအဆင့်ဆင့်
မိမိ Repository ထဲက ရွှေ့ချင်တဲ့ CSV ဖိုင်တစ်ခုခုကို နှိပ်ပြီး ဝင်လိုက်ပါ (ဥပမာ - Level1A.csv)။

ဖိုင်ရဲ့ ညာဘက်အပေါ်နားမှာရှိတဲ့ ခဲတံပုံစံ (Edit this file) ခလုတ်ကို နှိပ်ပါ။

အပေါ်နားက ဖိုင်နာမည် ရိုက်တဲ့ကွက်လပ်ကို ကြည့်ပါ။ လက်ရှိမှာ Level1A.csv လို့ ရှိနေပါလိမ့်မယ်။

အဲဒီဖိုင်နာမည်ရဲ့ ရှေ့ဆုံးမှာ မိမိရွှေ့ချင်တဲ့ Folder နာမည်ဖြစ်တဲ့ 2025/ ကို ရိုက်ထည့်လိုက်ပါ။

(ဥပမာ - Level1A.csv မှ 2025/Level1A.csv သို့ ပြောင်းသွားပါလိမ့်မယ်)။

လမ်းကြောင်း ပြောင်းသွားပြီဆိုရင် ညာဘက်အပေါ်က "Commit changes..." (စိမ်းပြာရောင်ခလုတ်) ကိုနှိပ်ပြီး အတည်ပြုလိုက်ပါ။

ဒါဆိုရင် အဲဒီဖိုင်ဟာ 2025 ဆိုတဲ့ ဖိုဒါအသစ်ထဲကို အလိုအလျောက် ရောက်ရှိသွားမှာ ဖြစ်ပါတယ်။ ကျန်တဲ့ Headmaster.csv၊ Teachers.csv စတဲ့ ဖိုင်တွေကိုလည်း ဒီနည်းလမ်းအတိုင်း လမ်းကြောင်းရှေ့မှာ 2025/ ခံပြီး တစ်ခုချင်းစီ ရွှေ့ပေးနိုင်ပါတယ်ခင်ဗျာ။
